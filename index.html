最长回文子串

发表于 2019-03-26 | 分类于 LeetCode

问题描述

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

示例 1：

输入: “babad”
输出: “bab”
注意: “aba” 也是一个有效答案。
示例 2：

输入: “cbbd”
输出: “bb”

解决方案

暴力方法（时间复杂度是O（n^3））：两层for循环，第一层是i为下标，第二层以j=i+1为始点,判断i以后的串是否存在回文串，当发现s[i]==s[j]时说明有回文的可能，这时将范围缩小到[i,j]之间，以k属于[i+1,(i+j-1)/2],t属于[j-((i+j-1)/2),j-1],比较s[k]与s[t]是否相等（其实就是判断对称位置是否相同），只要有一个对不相等就不是回文串，就break出最近的for循环，当全部相等时，将长度与之前长度比较，大于时，更新count值，更新mark数组（返回字符串区间）。注意：s.substring（start，end），返回的区间是[start,end)。

中心扩展法(时间复杂度是O（n^2）)：将原有的字符串用“#”进行扩充，例如将s=“abas”变成str=“#a#b#a#s#”,然后从i=0开始遍历str串，寻找str[i]左右两边对称位置是否相同，若相同则count++;不相同break；将找到的最长回文子串长度，对应原来s中起初和终点下标记录下来。

动态规划法(时间复杂度是O（n^2）)：初始化Boolean数组f[len][len],对角元素为true，其他为false。当一个字母时它是回文子串；当两个字母时，s[i]=s[j]则就是回文子串；当大于两个字母时，s[i]=s[j]的同时s[i+1][j-1]子串也是回文时，[i,j]才是回文子串。

$f\left [ i,j \right ]=\left{\begin{matrix}true & , j=i\s[i]=s[j] & ,j=i+1\ s[i]=s[j] and f[i+1][j-1]=true& ,j>i+1\end{matrix}\right.$

分发饼干

发表于 2019-03-25 | 分类于 LeetCode

问题描述

假设你是一位很棒的家长，想要给你的孩子们一些小饼干。但是，每个孩子最多只能给一块饼干。对每个孩子 i ，都有一个胃口值 gi ，这是能让孩子们满足胃口的饼干的最小尺寸；并且每块饼干 j ，都有一个尺寸 sj 。如果 sj >= gi ，我们可以将这个饼干 j 分配给孩子 i ，这个孩子会得到满足。你的目标是尽可能满足越多数量的孩子，并输出这个最大数值。

注意：

你可以假设胃口值为正。
一个小朋友最多只能拥有一块饼干。

示例 1:

输入: [1,2,3], [1,1]

输出: 1

解释:
你有三个孩子和两块小饼干，3个孩子的胃口值分别是：1,2,3。
虽然你有两块小饼干，由于他们的尺寸都是1，你只能让胃口值是1的孩子满足。
所以你应该输出1。
示例 2:

输入: [1,2], [1,2,3]

输出: 2

解释:
你有两个孩子和三块小饼干，2个孩子的胃口值分别是1,2。
你拥有的饼干数量和尺寸都足以让所有孩子满足。
所以你应该输出2.

解决方案

先把孩子胃口和饼干数组从小到大排列，然后用小值得饼干去喂胃口小的孩子，这样可以让更多的小孩得到满足。具体看代码注解。

两数相加

发表于 2019-03-25 | 分类于 LeetCode

问题描述

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

解决方案

从低位开始相加，如有进位则加到上一位（而且进位只能是1），用sum来存储每位上的和，将sum%10作为新链表的value值，将sum/10当做进位值再赋给sum,下一位相加时直接加到sum里。当其中一个链表短于另一个链表或者最高位有进位时，需要新建一个节点。

寻找两个有序数组的中位数

发表于 2019-03-19 | 分类于 LeetCode

问题描述

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 O(log(m + n))。

你可以假设 nums1 和 nums2 不会同时为空。

解题思路

不需要真的把两个数组有序合并在一起再算中位数。首先计算两个数组长度和sum，如果是偶数，那么中位数就是下标为(sum-1)/2和sum/2的数的和除以2.0，先用count计数排序两个数组的数，直到count==(sum-1)/2和sum/2，取出相应的中间数，除以2.0就可以得到想要的double类型的中位数；如果是奇数，中位数就是下标为sum/2的数，还是用count来计数，排序两个数组里的值，直到count==sum/2，取出中位数强转为double。

如何解决失眠问题

前言

没想到呀？我的第一个博客文章竟然是关于失眠的问题，哈哈哈，有意思呢~因为最近睡眠质量不是很好，晚上很困却又睡不着，这种感觉很折磨人，所以我就发明了如下方法，我觉得对我来说很管用，希望对你也一样~让我们一起来香香的睡觉叭

怎么睡得着

1.洗脸、刷牙、用热温水泡脚（或者洗个热水澡）
2.晚上9点半躺在床上
3.安静的看30分钟纸质书（内容不要对情绪有太大浮动的书）
4.10点整，合上书，躺在床上，睡觉之前一定要把枕头摆放到最舒适的状态
5.平躺着，让身体与床的接触面积最大，开始接下来的心理暗示
6.首先，让眼睛、鼻子、面部、手指、脚趾等身体的各个部位尽可能处在最放松的状态，不断的让身体放松，放松再放松，然后达到一种境界，你的头，两个臂膀、两条腿处在“消失”的状态，然后想象自己在大自然里，身边鸟语花香，溪水潺潺…最后飘在一朵云上，很轻很轻，在这个过程最好周围环境是安静的没有太大的晃动，连呼吸都要轻轻的…当你觉得对周围的意识有点恍惚时，你现在就处在浅睡眠，这个时候你就可以换一个你平时最舒服的睡姿（不受束缚的那种）,这样80%不会因为睡不着翻来覆去的了，如果真有的话，那就再对自己施一遍“魔法”。
7.再睁眼就是闹铃在呼唤你~然后你就处在 紧张地睡觉 （大脑呼唤自己赶快醒过来，可实力不允许，身体黏在床上，意识模糊，心情紧张，迟到要点名or请病假要扣工资）状态~哈哈哈嗝

&心小编&祝大家永远是一颗睡的香的太阳花~笔芯
