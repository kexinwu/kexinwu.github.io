寻找两个有序数组的中位数

发表于 2019-03-19 | 分类于 Leetcode

问题描述

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 O(log(m + n))。

你可以假设 nums1 和 nums2 不会同时为空。

解题思路

不需要真的把两个数组有序合并在一起再算中位数。首先计算两个数组长度和sum，如果是偶数，那么中位数就是下标为(sum-1)/2和sum/2的数的和除以2.0，先用count计数排序两个数组的数，直到count==(sum-1)/2和sum/2，取出相应的中间数，除以2.0就可以得到想要的double类型的中位数；如果是奇数，中位数就是下标为sum/2的数，还是用count来计数，排序两个数组里的值，直到count==sum/2，取出中位数强转为double。

如何解决失眠问题

发表于 2019-02-27

前言

没想到呀？我的第一个博客文章竟然是关于失眠的问题，哈哈哈，有意思呢~因为最近睡眠质量不是很好，晚上很困却又睡不着，这种感觉很折磨人，所以我就发明了如下方法，我觉得对我来说很管用，希望对你也一样~让我们一起来香香的睡觉叭

怎么睡得着

1.洗脸、刷牙、用热温水泡脚（或者洗个热水澡）
2.晚上9点半躺在床上
3.安静的看30分钟纸质书（内容不要对情绪有太大浮动的书）
4.10点整，合上书，躺在床上，睡觉之前一定要把枕头摆放到最舒适的状态
5.平躺着，让身体与床的接触面积最大，开始接下来的心理暗示
6.首先，让眼睛、鼻子、面部、手指、脚趾等身体的各个部位尽可能处在最放松的状态，不断的让身体放松，放松再放松，然后达到一种境界，你的头，两个臂膀、两条腿处在“消失”的状态，然后想象自己在大自然里，身边鸟语花香，溪水潺潺…最后飘在一朵云上，很轻很轻，在这个过程最好周围环境是安静的没有太大的晃动，连呼吸都要轻轻的…当你觉得对周围的意识有点恍惚时，你现在就处在浅睡眠，这个时候你就可以换一个你平时最舒服的睡姿（不受束缚的那种）,这样80%不会因为睡不着翻来覆去的了，如果真有的话，那就再对自己施一遍“魔法”。
7.再睁眼就是闹铃在呼唤你~然后你就处在 紧张地睡觉 （大脑呼唤自己赶快醒过来，可实力不允许，身体黏在床上，意识模糊，心情紧张，迟到要点名or请病假要扣工资）状态~哈哈哈嗝

&心小编&祝大家永远是一颗睡的香的太阳花~笔芯
