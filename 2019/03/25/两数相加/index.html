两数相加

发表于 2019-03-25 | 分类于 LeetCode

问题描述

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

解决方案

从低位开始相加，如有进位则加到上一位（而且进位只能是1），用sum来存储每位上的和，将sum%10作为新链表的value值，将sum/10当做进位值再赋给sum,下一位相加时直接加到sum里。当其中一个链表短于另一个链表或者最高位有进位时，需要新建一个节点。
