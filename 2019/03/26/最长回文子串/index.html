最长回文子串

发表于 2019-03-26 | 分类于 LeetCode

问题描述

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

示例 1：

输入: “babad”
输出: “bab”
注意: “aba” 也是一个有效答案。
示例 2：

输入: “cbbd”
输出: “bb”

解决方案

暴力方法（时间复杂度是O（n^3））：两层for循环，第一层是i为下标，第二层以j=i+1为始点,判断i以后的串是否存在回文串，当发现s[i]==s[j]时说明有回文的可能，这时将范围缩小到[i,j]之间，以k属于[i+1,(i+j-1)/2],t属于[j-((i+j-1)/2),j-1],比较s[k]与s[t]是否相等（其实就是判断对称位置是否相同），只要有一个对不相等就不是回文串，就break出最近的for循环，当全部相等时，将长度与之前长度比较，大于时，更新count值，更新mark数组（返回字符串区间）。注意：s.substring（start，end），返回的区间是[start,end)。

中心扩展法(时间复杂度是O（n^2）)：将原有的字符串用“#”进行扩充，例如将s=“abas”变成str=“#a#b#a#s#”,然后从i=0开始遍历str串，寻找str[i]左右两边对称位置是否相同，若相同则count++;不相同break；将找到的最长回文子串长度，对应原来s中起初和终点下标记录下来。

动态规划法(时间复杂度是O（n^2）)：初始化Boolean数组f[len][len],对角元素为true，其他为false。当一个字母时它是回文子串；当两个字母时，s[i]=s[j]则就是回文子串；当大于两个字母时，s[i]=s[j]的同时s[i+1][j-1]子串也是回文时，[i,j]才是回文子串。
